# Port Said Transit GTFS

[![License: CC BY-NC 4.0](https://img.shields.io/badge/License-CC%20BY--NC%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc/4.0/)

GIS & GTFS public transit data for **Port Said Governorate, Egypt**, covering local public transportation services within the city.

This project aims to document, structure, and publish public transport data in Port Said in an open, standardized format to support mapping platforms, researchers, developers, and civic tech initiatives.

---

## GeoJSON

Geospatial data representing public transport routes and stops in **GeoJSON** format.

The GeoJSON files include:

* Route geometries (LineString / MultiLineString)
* Stop locations (Point features)
* Properties structured for easy use in web maps and GIS tools

The data can be directly used with tools and libraries such as:

* Leaflet
* Mapbox
* OpenLayers
* QGIS
* Any GeoJSON-compatible mapping platform

---

## GTFS

A working **GTFS feed** describing public transport services in Port Said, including:

* Routes and shapes
* Trips and schedules
* Stops (including practical representations for flexible boarding locations)
* Service calendars

The feed is designed to be compatible with GTFS validators and mapping platforms such as Google Maps, while reflecting the real-world operational nature of public transport in the city.

## Disclaimer

This dataset reflects on-the-ground public transport operations, which may include flexible boarding and alighting practices not strictly aligned with fixed-stop systems. The data is provided for informational and mapping purposes and may evolve over time.
